# Basics

- `C-h i`: Open info
- `C-h i m <manual> RET`: Open specific manual.

## Info shortcuts

- `i <index>`: Go to index entry.
- `n`: Next page.
- `p`: Previous page.

To open a specific section, use the `info` function:

```
M-: (info "(elisp)Variable Definitions") RET1
```
