# Overview

## Flags

- `N`: New
- `P`: Passed (Forwarded)
- `R`: Replied
- `S`: Seen
- `T`: Thrashed
- `a`: has attachments
- `u`: unread
- `x`: encrypted
- `s`: signed

Other commands:

- `C-+`: Increase the number of visible rows.
- `C--`: Reduce the number of visible rows.


# Mail Reading Pane

- `g` : Go to URL using the number of the link.
