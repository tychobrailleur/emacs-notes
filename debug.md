# IELM

- `M-x ielm`: Start interactive emacs lisp


# edebug

- `C-u C-M-x`: `eval-defun` with debug flag.  Start debbuging.
- `n`: Next step
- `SPC`: Next step
- `e`: Evaluate expression
- `b`: Add breakpoint
- `g`: Execute until next breakpoint
- `q`: Quit debugging
